# InUni - MVP для Хакатона

## 🚀 Быстрый старт

```bash
# 1. Запуск
docker-compose up -d

# 2. Открыть http://localhost
# 3. Готово!
```

## 📝 Основные фичи

### Регистрация и авторизация
- Регистрация по email
- Хеширование паролей SHA-256
- Сохранение в PostgreSQL

### Профили
- Редактирование профиля
- Загрузка фото
- Навыки и направление
- AI-анализ профиля (Groq API)

### Команды
- Создание команд (хакатоны, проекты, стартапы)
- Поиск по названию
- Фильтрация по категориям
- Демо-данные при первом запуске

### Заявки
- Отправка заявок в команды
- Принятие/отклонение заявок
- Статусы в реальном времени

### Чат
- Командный чат
- Сохранение сообщений в БД
- Пагинация для масштабируемости

## 🛠 Технологии

| Компонент | Технология |
|-----------|------------|
| Frontend | Vue 3 + Pinia |
| Backend | Node.js + Express |
| Database | PostgreSQL 16 |
| AI | Groq API (Llama 3) |
| DevOps | Docker Compose |

## 🔌 API Endpoints

### Auth
- `POST /api/auth/register` - Регистрация
- `POST /api/auth/login` - Логин

### Profiles
- `GET /api/profiles/:id` - Получить профиль
- `PUT /api/profiles/:id` - Обновить профиль
- `POST /api/profiles/:id/photo` - Загрузить фото

### Teams
- `GET /api/teams?q=&category=` - Список команд (с поиском)
- `POST /api/teams` - Создать команду
- `DELETE /api/teams/:id` - Удалить команду

### Applications
- `POST /api/applications` - Отправить заявку
- `GET /api/applications/owner/:id` - Входящие заявки
- `GET /api/applications/user/:id` - Исходящие заявки
- `PUT /api/applications/:id` - Обновить статус

### Messages
- `GET /api/messages/:teamId` - Сообщения команды
- `POST /api/messages` - Отправить сообщение

## 🐳 Структура Docker

```yaml
services:
```
